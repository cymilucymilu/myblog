# Hello World

欢迎来到我的博客！这是我使用 Next.js 15 + MDX 搭建的个人博客。

## 特性

- **Next.js 15** - 最新的 React 框架
- **MDX** - 支持 JSX 的 Markdown
- **Tailwind CSS** - 实用优先的 CSS 框架
- **Shadcn/UI** - 高质量的 UI 组件
- **Fuse.js** - 客户端全文搜索
- **Giscus** - 基于 GitHub Discussions 的评论系统

## 代码示例

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```

## 开始使用

1. 克隆仓库
2. 安装依赖：`npm install`
3. 运行开发服务器：`npm run dev`
4. 在 `content/posts/` 目录下创建你的文章
