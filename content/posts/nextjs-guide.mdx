# Next.js 15 新特性详解

Next.js 15 带来了许多令人兴奋的改进和新特性。

## App Router 稳定版

App Router 现已稳定，提供了：

- 嵌套布局
- 错误处理
- 加载状态
- 并行路由
- 拦截路由

## React 19 支持

Next.js 15 完全支持 React 19，包括：

- Server Components
- Server Actions
- Suspense 改进

## 性能优化

- 更快的构建速度
- 更小的 bundle 体积
- 改进的缓存策略

```javascript
// Server Action 示例
async function createPost(formData: FormData) {
  'use server';
  // 直接操作数据库
}
```

## 总结

Next.js 15 是构建现代 Web 应用的最佳选择之一。
